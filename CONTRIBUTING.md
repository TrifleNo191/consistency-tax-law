# How to Contribute

## Contribution Types
1. **Validation**: Replicate experiments
2. **Refutation**: Find errors
3. **Extension**: Apply to new domains
4. **Implementation**: Build tools

## Recognition
- Validation → Co-authorship
- Refutation → Co-authorship  
- Extension → Co-authorship
- Implementation → Acknowledgment + tool credit

## Getting Started
1. Read [paper.pdf]
2. Check [OPEN_QUESTIONS.md]
3. Pick a contribution type
4. Open an issue or PR
